# arrow functions

**arrow functions** are a more concise syntax for writing function expressions.

```javascript
// (param1, param2, paramN) => expression

// ES5
var multiplyES5 = function(x, y) {
  return x * y;
};

// ES6
const multiplyES6 = (x, y) => { return x * y };
```
